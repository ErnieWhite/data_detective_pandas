# Data Detective: Pandas Edition

A playable command-line game that teaches **pandas** by having you solve data puzzles.

## What you will learn
- Reading CSV files and inspecting frames
- Filtering rows and selecting columns
- Cleaning messy data (`fillna`, `str.strip`, `normalization`)
- Aggregations (`group by`, `mean`, `sum`)
- Data parsing, sorting, and time filtering
- Merging and join indicators
- Pivot tables and reshaping
- Regex extraction with `str.extract`
- Rolling windows and time series analysis
- Duplicate detection and ranking
- Chucked processing for large files
- A final boss case that combines multiple techniques

## Run the game
```bash
python main.py
